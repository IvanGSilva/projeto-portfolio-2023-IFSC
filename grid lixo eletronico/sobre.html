SOBRE O SITE


Este projeto foi construído para uma avaliação de portfólio para o Curso Superior de Sistemas e Tecnologias para Internet no IFSC - Câmpus Garopaba SC no ano de 2023 para o 3° Semestre


Para a construção do site foram usadas as técnicas apredidas nas aulas e provenientes de pesquisa própria como:

  • Flexbox;

  • Grid;

  • Media Query;

  • Var();

  • Transition;

  • Emmet

  • Entre outros...
